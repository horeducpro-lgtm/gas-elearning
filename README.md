# GAS – Global Academy of Skills

Plateforme e‑Learning moderne et robuste (Next.js + FastAPI + PostgreSQL + Redis).
- Frontend: Next.js 15, TypeScript, Tailwind, NextAuth
- Backend: FastAPI, SQLAlchemy, Alembic, Celery
- DB: PostgreSQL, Redis (tâches asynchrones)
- Infra: Docker, CI/CD

## Démarrage rapide
1. Mettre à jour WSL: `wsl --update`
2. Lancer Docker Desktop.
3. Créer le frontend (voir section frontend).
4. Créer le backend (voir section backend).
5. `docker compose up -d` pour DB/Redis.
